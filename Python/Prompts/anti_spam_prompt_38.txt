### Критерии SPAM (маркировать если есть ХОТЯ БЫ ОДИН признак):
1. **Финансовые предложения + контакт:**
   - Упоминание сумм ($, ₽, "доход", "прибыль", "заработок", "пассивный доход") + ЛЮБОЙ призыв к действию 
     ("пиши", "ЛС", "напиши", "жду", "расскажу детали", "обсудить", "подробности", "интересно узнать?", "подробнее", "рад(а) обсудить", "напишите мне", "жду в ЛС", "отвечай", "плюс в ЛС", "поделюсь подробностями")
   - *Примеры:*
     ▸ "Доход 110$/день → напиши"
     ▸ "Хочешь $70/день? Отвечай"

2. **Гарантии быстрой выгоды:**
   - Любые комбинации (включая разделение на строки): 
     - "без вложений"/"без опыта" + ("требуются люди"/"набор"/"несколько человек"/"[число] человек")
     - "всё просто"/"без стрессов"/"без сложностей" + (цифры/$/₽/проценты)
     - "пассивный доход"/"забирай [X]% прибыли"/"формат [X/X]"
   - *Примеры:*
     ▸ "Нужно 5 человек → 80% прибыли"
     ▸ "Без сложностей: зови друзей + $600/неделя"

3. **Шаблонные структуры (включая многострочные):**
   - [Приветствие] + [доход/валюта/проценты] + [контакт]
   - Упоминание "партнёрств", "форматов", "направлений", "схем" с цифрами (даже без валюты)
   - *Примеры:*
     ▸ "Добрый день! $250/день\nПиши 'Да'"
     ▸ "Партнёрство 80/20 → ваш доход"

4. **Скрытые маркеры:**
   - Связка ("удалёнка"/"смартфон"/"гибкий график"/"онлайн") + (валюта/проценты) + контакт
   - Любые манипуляции: "Срочно!", "Только сегодня", "Уникальный шанс", "Хватит...", "Не упусти", "Подробности в личке"

5. **Многострочный спам:**
   - Финансовые термины и контактные фразы в РАЗНЫХ строках
   - *Пример:*
     ▸ "Удалёнка без начальников\nОт $13/час\nЖду в ЛС"

### Уточнения:
- **Цифры в любом формате:** "$600/неделя", "70$", "250 долларов", "80%", "X/X"
- **Контактные фразы:** Даже косвенные ("буду рада обсудить", "если заинтересовало", "расскажу в ЛС") → SPAM
- **Доход без суммы:** "пассивный доход", "прибыль", "выгода" + контакт → SPAM
- **Проценты + контакт:** "забирай 80%", "ваши 80%" → SPAM
- **Многострочность:** Учитывать связки даже при переносах строк

### Критерии NOT_SPAM (только при ПОЛНОМ отсутствии признаков выше):
- **Технические термины:** 
  ▸ Квантование ("Q4_K_M"), fine-tuning, vLLM, RAG, векторные базы
- **Рабочие/личные темы:** 
  ▸ Встречи ("в 18:00"), трек-номера, гипотезы об ИИ
  ▸ Вопросы без финансов: "Как думаете...", "Был похожий случай..."
- **Аналитика ИИ:** 
  ▸ Обсуждение поведения ChatGPT, этики, локальных моделей

### Автоматические правила:
1. **Любая связка** ("$"/"₽"/"доллар"/"%") + ("пиши"/"ЛС"/"жду") → SPAM (даже через переносы строк)
2. **Любое** "партнёрство"/"формат"/"направление" + цифры → SPAM
3. **Игнорировать:** 
   - Опечатки, длину текста, смайлы
   - Обсуждение математики/университетов без финансовых терминов
