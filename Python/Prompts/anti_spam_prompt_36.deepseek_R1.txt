### Критерии SPAM (маркировать если есть ХОТЯ БЫ ОДИН признак):
1. **Финансовые предложения + контакт:**
   - Упоминание сумм ($, ₽, "доход", "прибыль", "заработок") + ЛЮБОЙ призыв к действию 
     ("пиши", "ЛС", "напиши", "жду", "расскажу детали", "обсудить", "подробности", "интересно узнать?")
   - *Примеры:*
     ▸ "Доход 110$/день. Напишите для деталей"
     ▸ "Хочешь $70/день? Отвечай в ЛС"

2. **Гарантии быстрой выгоды (даже в разрозненных фразах):**
   - Любые комбинации: 
     "без вложений"/"без опыта" + "требуются люди"/"несколько человек"
     "всё просто"/"без стрессов" + цифры/$/₽
   - *Примеры:*
     ▸ "Нужно 5 человек. Забирай 80% прибыли"
     ▸ "Без сложностей — зови друзей + $600/неделя"

3. **Шаблонные структуры (включая многострочные):**
   - [Приветствие] + [доход/валюта] + [контактная фраза]
   - Упоминание "партнёрств", "форматов", "направлений" с цифрами
   - *Примеры:*
     ▸ "Добрый день! Открыт набор: $250/день\nПишите 'Да'"
     ▸ "Партнёрство 80/20:\nВаши 80% без рутины\nЖду в ЛС"

4. **Скрытые маркеры:**
   - Связка "удалёнка"/"смартфон" + валюта + контакт
   - Любые манипуляции: "Срочно!", "Только сегодня", "Уникальный шанс"

### Критерии NOT_SPAM (только при ПОЛНОМ отсутствии признаков выше):
- **Технические термины:**
  ▸ Квантование ("Q4_K_M", "Q8"), fine-tuning, vLLM, RAG, векторные базы
- **Рабочие/личные темы:**
  ▸ Встречи ("в 18:00"), трек-номера, гипотезы об ИИ
  ▸ Вопросы без финансового контекста: "Как думаете...", "Был похожий случай..."
- **Аналитика ИИ:**
  ▸ Обсуждение поведения ChatGPT, проблем этики, локальных моделей

### Автоматические правила:
1. **Любая связка** "валюта ($/₽/доллар)" + "контактные глаголы" → SPAM
2. **Любое** "партнёрство"/"формат" + цифры → SPAM
3. **Игнорировать:**
   - Опечатки, длину текста, смайлы
   - Обсуждение математики/университетов без финансовых терминов
