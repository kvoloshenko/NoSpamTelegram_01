### Критерии SPAM (маркировать если есть ХОТЯ БЫ ОДИН признак):
1. **Финансовые предложения + контакт:**
   - Упоминание сумм ($, ₽, "доход", "прибыль", "заработок", "пассивный доход") + ЛЮБОЙ призыв к действию 
     ("пиши", "ЛС", "напиши", "жду", "расскажу детали", "обсудить", "подробности", "интересно узнать?", "подробнее", "рад(а) обсудить", "напишите мне", "жду в ЛС", "отвечай")
   - *Примеры:*
     ▸ "Доход 110$/день. Напишите для деталей"
     ▸ "Хочешь $70/день? Отвечай в ЛС"

2. **Гарантии быстрой выгоды:**
   - Любые комбинации (даже в разных строках): 
     - "без вложений"/"без опыта" + "требуются люди"/"набор"/"несколько человек"/"5 человек"
     - "всё просто"/"без стрессов"/"без сложностей" + цифры/$/₽
     - "пассивный доход"/"забирай X% прибыли"
   - *Примеры:*
     ▸ "Нужно 5 человек. Забирай 80% прибыли"
     ▸ "Без сложностей — зови друзей + $600/неделя"

3. **Шаблонные структуры (включая многострочные и разделенные текстом):**
   - [Приветствие] + [доход/валюта] + [контактная фраза]
   - Упоминание "партнёрств", "форматов", "направлений", "схем" с цифрами (даже без явного указания валюты)
   - *Примеры:*
     ▸ "Добрый день! Открыт набор: $250/день\nПишите 'Да'"
     ▸ "Партнёрство 80/20: ваш результат без рутины"

4. **Скрытые маркеры:**
   - Связка "удалёнка"/"смартфон"/"гибкий график" + валюта + контакт
   - Любые манипуляции: "Срочно!", "Только сегодня", "Уникальный шанс", "Хватит...", "Не упусти"

### Уточнения:
- **Цифры в любом формате:** "$600/неделя", "70$", "250 долларов", "80%"
- **Контактные фразы:** Даже косвенные ("буду рада обсудить", "если заинтересовало") → SPAM
- **Партнёрство/формат:** Достаточно упоминания с цифрами (80/20, 60/40) → SPAM
- **Доход без суммы:** "пассивный доход", "прибыль", "выгода" + контакт → SPAM

### Критерии NOT_SPAM (только при ПОЛНОМ отсутствии признаков выше):
- **Технические термины:**
  ▸ Квантование ("Q4_K_M", "Q8"), fine-tuning, vLLM, RAG, векторные базы
- **Рабочие/личные темы:**
  ▸ Встречи ("в 18:00"), трек-номера, гипотезы об ИИ
  ▸ Вопросы без финансового контекста: "Как думаете...", "Был похожий случай..."
- **Аналитика ИИ:**
  ▸ Обсуждение поведения ChatGPT, проблем этики, локальных моделей

### Автоматические правила:
1. **Любая связка** "валюта ($/₽/доллар)" + "контактные глаголы" → SPAM (даже через другие слова)
2. **Любое** "партнёрство"/"формат"/"направление" + цифры → SPAM
3. **Игнорировать:**
   - Опечатки, длину текста, смайлы
   - Обсуждение математики/университетов без финансовых терминов
